# **App Name**: QuickLoan

## Core Features:

- Loan Application Submission: Allow users to submit loan applications with necessary details.
- Application Status Tracking: Enable users to track the status of their loan applications.
- Loan Details Viewing: Display details of approved loans to the users.
- Profile and Application History: Allow users to access their profile and application history.
- Admin Dashboard: Provide an admin interface for approving/rejecting loan applications.
- Loan Plan Listings Management: Enable admins to add and manage loan plan listings.
- Fraud Detection: Use an AI fraud detection tool to assess new loan applications

## Style Guidelines:

- Primary color: Professional green (#4CAF50) for trust and stability.
- Background color: Off-white (#F0F0F0), providing a clean, modern feel. The hue is subtly similar to green, but very desaturated.
- Accent color: Blue (#2196F3) for interactive elements and highlights, conveying reliability.
- Body and headline font: 'PT Sans', a humanist sans-serif, for a blend of modernity and readability.
- Modern UI layout with rounded corner buttons and card-based design.
- Simple, professional icons for key actions and information display.
- Subtle transitions and animations for a smooth user experience.
